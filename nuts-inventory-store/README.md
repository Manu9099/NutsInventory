# NutsInventory Store

Storefront cliente de NutsInventory construido con React + TypeScript + Vite.

## Qué incluye

- Home comercial
- Catálogo de productos
- Detalle de producto
- Carrito de compras
- Checkout base
- Login de cliente
- Perfil
- Historial de pedidos

## Stack

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- Axios
- Framer Motion
- Tailwind CSS
